# Dataset

This project uses a YouTube trending videos dataset named:

`youtube.csv`

The complete dataset is not included in this repository.

To run the notebook:

1. Download the dataset from the original source.
2. Upload the file to Google Colab or Jupyter Notebook.
3. Update the dataset path in the notebook.
4. Make sure the file is named `youtube.csv`.

```python
import pandas as pd

df = pd.read_csv("youtube.csv")
